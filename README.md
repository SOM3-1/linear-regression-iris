# Linear Regression Classification Project

## How to Run

This project requires Python 3.

---

## Steps

### macOS / Linux:
```bash
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
python src/linear_regression.py
deactivate
```
> If python3 is not available, use the Python command installed on your system
### Windows:
```bash
python3 -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
python src/linear_regression.py
deactivate
```
> If python3 is not available, use the Python command installed on your system

## Output

- Training and testing results are printed in the console

- Generated plots are saved in the images/ directory

- Project report is provided as Project_report_1002246620.docx in the project root directory
